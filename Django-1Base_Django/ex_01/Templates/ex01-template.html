Ex01: Template engine.

Django's default template system is the Django Template Language (DTL). It renders text files with template tags and variables into final HTML.

Blocks

Blocks define replaceable sections in a base template. Child templates use {% verbatim %}{% block ... %}{% endblock %}{% endverbatim %} to override those sections.

Loops: for ... in

    {% for feature in features %}
  • {{ feature }}
    • {% endfor %}

If control structures

{% if show_note %}

The if tag renders content only when its condition is true.

{% else %}

The condition is false, so this alternative branch is rendered.

{% endif %}

Displaying context variables

Variables passed by the view are displayed with double braces, for example: {% verbatim %}{{ engine_name }}{% endverbatim %}.
