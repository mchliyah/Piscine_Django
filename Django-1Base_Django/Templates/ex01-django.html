Ex01: Django, framework web.

Django is a high-level Python web framework created to help developers build robust, secure websites quickly.

It was started in 2003 inside the Lawrence Journal-World newsroom by Adrian Holovaty and Simon Willison, then released publicly in 2005. Django follows practical principles like reusable components, explicit configuration, and the DRY philosophy.

Today, Django is maintained by the Django Software Foundation and is widely used for content-heavy and data-driven web platforms.
