Ex00: Markdown Cheatsheet.

A compact reference for common Markdown syntax.

Headings

# H1
## H2
### H3
#### H4
##### H5
###### H6

Text Emphasis

  • *italic* or _italic_
  • **bold** or __bold__
  • ***bold italic***
  • ~~strikethrough~~

Paragraphs and Line Breaks

Separate paragraphs with a blank line.

For a line break, end a line with two spaces or use <br>.

Blockquotes

> This is a blockquote.
>> Nested blockquote.
This is a rendered quote example.

Lists

Unordered

- Item A
- Item B
  - Nested item

Ordered

1. First
2. Second
3. Third

Task List (GitHub Flavored Markdown)

- [x] Done
- [ ] To do

Links and Images

[OpenAI](https://openai.com)
![Alt text](https://example.com/image.png "Optional title")

Code

Inline Code

Use backticks like `print("hello")`.

Code Block

```python
def greet(name):
	return f"Hello, {name}"
```

Horizontal Rule

---
***
___

Tables (GitHub Flavored Markdown)

| Name | Role |
|------|------|
| Ada  | Dev  |
| Lin  | Ops  |
Name Role
Ada Dev
Lin Ops

Escaping Characters

Use backslash to escape Markdown syntax:

\*not italic\*

HTML in Markdown

Markdown parsers usually accept inline HTML:

<mark>highlight</mark>
<sub>subscript</sub>
<sup>superscript</sup>
