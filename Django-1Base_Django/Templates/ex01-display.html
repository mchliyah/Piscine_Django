Ex01: Display process of a static page.

  1. The browser sends an HTTP request to a URL such as /ex01/display.
  2. Django URL routing matches this request to a view function.
  3. The view executes and asks the template engine to render a template.
  4. The template is combined with optional context data to produce HTML.
  5. Django returns an HTTP response containing that HTML.
  6. The browser receives the response and displays the static page.
